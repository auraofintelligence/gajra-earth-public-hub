Fork path

Use the patterns. Keep the claims clean.

Many repos can become GAJRA-specific forks later. For now, this map shows working references and live neighbours.

Use now

Link to the current repo or public page when it already helps a visitor act, understand or verify.

Fork later

Make GAJRA-specific versions when the workflow needs its own audience, data model, copy or public boundary.

Label honestly

Archive, research, public prototype and working repo are different states. The page must say which is which.
